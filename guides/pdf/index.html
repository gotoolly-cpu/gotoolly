Frequently Asked Questions

How do I merge multiple PDFs into one?

Use the PDF Merger tool. Upload your PDFs in any order, drag to rearrange them, and click Merge. The combined file downloads instantly with no server uploads.

Can I convert Excel to PDF without losing formatting?

Yes. The Excel to PDF converter preserves tables, fonts, and layout. You can select specific sheets and choose page orientation before converting.

How do I unlock a password-protected PDF?

Use the Unlock PDF tool. Enter the document password and the tool removes the protection. All processing happens locally in your browser.

Can I add a watermark to a PDF without uploading it?

Yes. The PDF Watermark tool processes everything locally in your browser. You can add text watermarks with custom opacity, rotation, font, and color — no files are ever uploaded.

What is PDF OCR and how does it work?

PDF OCR (Optical Character Recognition) extracts text from scanned PDF documents using browser-based OCR technology. It converts image-based text into selectable, searchable content — all processed locally.
