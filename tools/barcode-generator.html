Frequently Asked Questions

What barcode formats are supported?

The tool supports Code 128, Code 39, EAN-13, EAN-8, UPC-A, and UPC-E barcode formats. Each format has specific encoding rules optimized for different use cases.

Can I download the barcode as an image?

Yes. You can download the generated barcode as PNG (raster) or SVG (vector). SVG is ideal for printing and scaling without quality loss.

What is the difference between Code 128 and Code 39?

Code 128 is more compact and supports the full ASCII character set. Code 39 is older and supports only uppercase letters, digits, and a few special characters.

Is my data uploaded to a server?

No. All barcode generation happens in your browser using the HTML5 Canvas API. Your data never leaves your device.
