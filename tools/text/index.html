Frequently Asked Questions

Are these text tools really free?

Yes, all 21 text tools are completely free to use. No sign-up, no usage limits, no premium tiers. Everything runs in your browser.

Can I compare two texts for differences?

Yes, the Text Diff tool compares two texts side by side and highlights additions, deletions, and changes with color-coded output.

Do you support Markdown editing and conversion?

Yes. The Markdown Editor provides a split-pane editor with live preview. You can also convert Markdown to HTML and HTML to Markdown with dedicated tools.

Can I minify HTML, CSS, and JavaScript?

Yes, we have dedicated minifiers for HTML, CSS, and JavaScript. Each removes comments, whitespace, and optimizes syntax for smaller file sizes.

What encoders and formatters are available?

We offer encoders for Base64, URL, and HTML entities, plus formatters for SQL, XML, and JSON, as well as a Regex Tester with live matching.

How these tools work: All processing happens in your browser. Files are not uploaded to servers. Nothing is logged or stored.
